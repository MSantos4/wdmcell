## Objetivo e Funcionalidades
O programa tem a finalidade de permitir a gestão de uma loja de celular. Ajudando com o controle de clientes, produtos e vendas.
Inicialmente contando com as principais funcionalidades, o programa é capaz de cadastrar os produtos, clientes e realizar as vendas.
